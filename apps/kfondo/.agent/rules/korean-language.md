# 언어 사용 규칙

1. **문서 및 아티팩트 작성**: 모든 `Implementation Plan`, `Task`, `Walkthrough` 등의 아티팩트 문서는 반드시 **한국어**로 작성해야 합니다.
2. **커밋 메시지**: Git 커밋 메시지는 **한국어**로 작성합니다.
3. **사용자 커뮤니케이션**: 사용자에게 보내는 메시지, 알림, 질문 등은 **한국어**를 사용합니다.
4. **코드 주석**: 코드 내의 주석도 가급적 한국어를 사용하여 작성하는 것을 권장합니다 (팀 컨벤션에 따름).
